# Vibe Coding Tetris Game

A classic Tetris game built using a vibe coding tool.
This project demonstrates AI-assisted, context-aware development.

## Features
- Falling tetromino blocks
- Keyboard controls (move, rotate, drop)
- Line clearing logic
- Score tracking
- Game over detection

## Technologies Used
- HTML
- CSS
- JavaScript
- Vibe coding tool (Replit Agent)
